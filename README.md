# SEO Sitemap Crawler — Streamlit Cloud

## Что загрузить в корень GitHub-репозитория

Замените старые файлы на эти:

- app.py
- requirements.txt
- runtime.txt

Главное: в requirements.txt больше нет lxml.

## Важно для Streamlit Cloud

Если приложение уже было создано, Python-версию лучше выбрать в интерфейсе Streamlit Cloud:

Manage app → Settings / Advanced settings → Python version → 3.12

После замены файлов:

1. Push в GitHub.
2. Streamlit Cloud → Manage app.
3. Clear cache.
4. Reboot app.
